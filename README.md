
# تحويل موديل Hermes إلى CoreML

## خطوات التشغيل

1. إنشاء بيئة جديدة:

```bash
python3 -m venv venv
source venv/bin/activate
```

2. تثبيت المتطلبات:

```bash
pip install -r requirements.txt
```

3. تشغيل التحويل:

```bash
python convert.py
```

> تأكد أنك متصل بالإنترنت لتحميل الموديل من Hugging Face.
